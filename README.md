# [情感的韧性——在风雨中共同成长](https://hoo.be/w-91wang)

没有一段感情会永远一帆风顺。外界的压力、生活的变故、内部的危机……这些风雨不是在摧毁关系，就是在锻造关系。区别在于，你们是否具备“情感的韧性”。

韧性，不是坚硬不摧，而是像竹子一样，懂得在风雨中弯曲，但根却紧紧相连。它意味着，在面对困难时，你们是站在一边，共同面对问题，而不是站在两边，互相成为问题。

培养韧性，需要共同创造美好的回忆作为情感储备，需要在每次克服小困难后积累信心，更需要一个坚定的信念：我们的关系，比出现的问题更重要。每一次共渡难关，都会在你们的关系史上留下一枚勋章，证明你们比想象中更强大。

最牢固的感情，并非从未经历黑暗，而是携手穿越了黑暗。
